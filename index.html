Projetos em Ciência de Dados

Construção de um programa de fidelidade com clusterização de clientes.

Utilizei Python, Estatística e técnicas não supervisionadas de Machine Learning para segmentar um grupo de clientes com base em suas características de compra. O objetivo era selecionar grupos para um programa de fidelidade, visando aumentar a receita da empresa. Se essa solução fosse implementada, o resultado esperado seria de $15 milhões em receita anual.

As ferramentas utilizadas foram:

  • Git, Gitlab e Github
  • Python, Pandas, Matplotlib e Seaborn
  • Jupyter Notebook
  • K-Means, Hierarquical Clustering, DBScan
  • AWS Cloud (EC2, S3, Postgres, SQLite)
  • Metabase Visualization
  • SAIBA MAIS

Ensaio de Machine Learning

Neste projeto, foram abordados os conceitos fundamentais do treinamento, ajuste de parâmetros e limites entre overfitting e underfitting de diversos algoritmos de Machine Learning nas tarefas de classificação, regressão e agrupamento, por meio de um ensaio de Machine Learning.

O resultado final do projeto foi um painel exibindo o desempenho desses algoritmos de Machine Learning, com base nas variações dos valores dos principais parâmetros de cada algoritmo, que regulam o estado de overfitting e underfitting.

As ferramentas utilizadas foram:

  • Python 3.10.4 e Scikit-learn
  • Algoritmos de classificação: KNN, Decision Tree, Random Forest e Logistic Regression
  • Algoritmos de regressão: Linear Regression, Decision Tree Regressor, Random Forest Regressor, Polinomial Regression, Linear Regression Lasso, Linear Regression Ridge, Linear Regression Elastic Net, Polinomial Regression Lasso, Polinomial Regression Ridge e Polinomial Regression Elastic Net
  • Algoritmos de agrupamento: K-Means e Affinity Propagation
  • Métricas de performance: Accuracy, Precision, Recall, F1-Score, R2, MSE, RMSE, MAE, MAPE e Silhouette Score
  • SAIBA MAIS
